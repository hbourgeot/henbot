## Commands for arithmetics operations

`/calc-sum X Y` = calculates the sum between the two elements

`/calc-sub X Y` = ccalculates the difference between the two elements

`/calc-mul X Y` = calculates the multiplication between the two elements

`/calc-div X Y` = calculates the division between the two elements

`/calc-mod X Y` = calculates the remainder of the division between the two elements

`/calc-perc X Y` = calculates the percentage X of Y

USAGE SYNTAX = `/calc-sum X Y`

X and Y are NUMBERS. Try NOT to place any letter or symbol near, if you enter more than two numbers, they will not be read
